# Photo Gallery

## Description:
This is an exercise to strengthen knowledge with NodeJS, creating a photo gallery where you can share different pictures that you like and want to show to your friends.

## Stack:

- Express
- MongoDB
- Mongoose
- Bootstrap
- Cloudinary
